# Sales Effectiveness Analysis

## Project Overview
This project analyzes sales leads data to improve sales effectiveness. 
The goal is to predict whether a lead is High Potential or Low Potential.

## Tools Used
- Python
- SQL
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn
- Jupyter Notebook

## Steps Performed
1. Data Cleaning
2. Handling Missing Values
3. Exploratory Data Analysis
4. Feature Engineering
5. Model Building
6. Prediction of Lead Potential

## Business Objective
To help sales teams identify high-potential leads before assigning sales agents.
